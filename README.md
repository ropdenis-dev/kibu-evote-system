# KIBU eVote - Blockchain-Powered Student Voting System

A decentralized voting platform for Kibabii University student elections, built on the Sepolia blockchain.

## Features
- Blockchain-verified voting
- MetaMask wallet integration
- Real-time results
- Secure student authentication
- Mobile-responsive design

## Technology Stack
- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Ethereum (Sepolia Testnet)
- Wallet: MetaMask
- Backend: Node.js/Express (coming soon)
- Database: MongoDB (coming soon)

## Setup Instructions
1. Clone the repository
2. Open index.html in your browser
3. Install MetaMask extension
4. Connect to Sepolia testnet
5. Register with your KIBU credentials
6. Start voting!

## File Structure
- `/css` - Stylesheets
- `/js` - JavaScript files
- `/data` - JSON mock data
- `index.html` - Landing page
- `login.html` - Login page
- `signup.html` - Registration
- `dashboard.html` - User dashboard
- `voting.html` - Voting interface
