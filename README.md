# KIBU eVote - Blockchain-Powered Student Voting System

A decentralized voting platform for Kibabii University student elections, built on the Sepolia blockchain.

## Features
- Blockchain-verified voting
- MetaMask wallet integration
- Real-time results
- Secure student authentication
- Mobile-responsive design

## Technology Stack
- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Ethereum (Sepolia Testnet)
- Wallet: MetaMask
- Backend: Node.js/Express
- Database: MongoDB 

## Setup Instructions
1. Install [MetaMask extension](https://metamask.io/download/)
2. Connect to Sepolia testnet
3. Visit [KIBU eVote](https://ropdenis-dev.github.io/kibu-evote-system/)
4. Register with your KIBU credentials
5. Start voting!

## File Structure
- `/css` - Stylesheets
- `/js` - JavaScript files
- `/data` - JSON mock data
- `index.html` - Landing page
- `login.html` - Login page
- `signup.html` - Registration
- `dashboard.html` - User dashboard
- `voting.html` - Voting interface
- `results.html` - Live results
- `verification.html` - My votes
- `admin.html` - Admin panel
## System Flow
Admin creates election > adds positions > adds candidates > starts election.

Student registers > logs in > selects candidates > submits to blockchain.

Results update in real-time with charts and analytics.
